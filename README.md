# 🚨 Explainable Fraud Intelligence Framework

## JP Morgan Data Analytics Virtual Experience (Forage)

An end-to-end fraud analytics solution that combines explainable machine learning, behavioural fraud intelligence, operational KPIs and interactive Power BI dashboards to support fraud investigators in prioritising high-risk alerts and improving operational decision-making.

---

## 🎯 Project Highlights

* Explainable Fraud Intelligence Framework
* Logistic Regression Operational Model
* XGBoost Challenger Model
* SHAP Explainability
* Fraud Alert Prioritisation
* Operational KPI Framework
* Interactive Power BI Dashboard
* Human-in-the-Loop Decision Support

---

## 🛠️ Technology Stack

* Python

* Git & GitHub
* VS Code

---

## 📊 Key Results

* **5,000** integrated banking transactions analysed
* **201** confirmed fraud cases (**4.02% fraud rate**)
* High-Risk alerts achieved a **12.02% fraud rate** compared with **2.41%** for Low-Risk alerts
* Logistic Regression selected as the operational model based on stronger fraud recall, explainability and governance suitability
* SHAP explainability used to improve transparency and investigator trust
