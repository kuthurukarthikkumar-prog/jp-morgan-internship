# Explainable Fraud Alert Prioritization and Investigation Control Tower

Company inspiration: JPMorganChase

## Package map
- `A_Main_Document/`: final student-facing DOCX project brief
- `B_Dataset/raw/`: raw source data
- `B_Dataset/processed/`: optional seed outputs
- `B_Dataset/metadata/`: data dictionary and schema notes
- `C_Supporting_Files/`: notebook, SQL, KPI list, guidance, templates

## Recommended start sequence
1. Read the project brief.
2. Review the data dictionary and schema note.
3. Open the starter notebook and SQL file.
4. Build your own analytic layer and outputs.
